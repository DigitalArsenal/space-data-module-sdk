# Module Publishing Examples

This directory shows the canonical `sdn-module` publication descriptor in the
two supported release modes:

- [standalone.sdn-module.json](./standalone.sdn-module.json)
- [attached.sdn-module.json](./attached.sdn-module.json)

See [docs/module-publication-standard.md](../../docs/module-publication-standard.md)
for the full cross-ecosystem publication rules and package-manager-specific
metadata carriers.
